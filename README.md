# Scatch
Scatch - Premium Bag E-commerce Website
Scatch is an e-commerce platform for selling premium bags. This single-owner website allows a seller to manage products and users to browse, add items to the cart, apply discount codes, and sort products based on various filters like date, discount, and new arrivals.

Features
Single-owner functionality: Admin dashboard for managing products (add, update, remove).
Product listing: Display of all premium bags with options to sort by date, discount, and new arrivals.
Cart management: Users can add products to the cart and manage them easily.
Discount codes: Functionality to apply discount codes during checkout.
Responsive design: Built using Tailwind CSS, optimized for mobile and desktop.

Tech Stack
Backend: Node.js, Express.js
Frontend: Tailwind CSS
Database: MongoDB
